# Emotion Detection Project

This project implements an AI-based Emotion Detection application
using the Watson NLP library and Flask web framework.

## Project Name
EmotionDetection

## Author
Syed Muneer

## Description
An application that detects emotions (anger, disgust, fear, joy, sadness)
from text input using IBM Watson NLP and deploys them via a Flask web server.

## Technologies Used
- Python
- Watson NLP Library
- Flask
- PyLint
